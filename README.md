# Cahier de Direction — PWA

## Déploiement GitHub Pages

1. Aller sur github.com → votre dépôt `cahier-direction`
2. Cliquer sur chaque fichier → "..." → "Edit" ou glisser-déposer les nouveaux fichiers
3. L'URL reste : `https://votre-pseudo.github.io/cahier-direction/`

## Mise à jour (nouvelle version)

1. Télécharger le nouveau ZIP
2. Dézipper
3. Sur GitHub : glisser-déposer les fichiers mis à jour directement dans le dépôt
4. Cliquer "Commit changes"
5. Attendre 1-2 minutes → le site se met à jour automatiquement

## Fichiers

- `index.html` — structure de la page
- `app.js` — toute la logique de l'application
- `style.css` — mise en forme
- `manifest.json` — config PWA (installation)
- `sw.js` — service worker (mode hors ligne)
