Chaque sauvegarde (💾) met à jour ce fichier automatiquement.
Pour synchroniser sur un autre appareil : ouvrez le cahier, cliquez ☁️ → Charger depuis OneDrive.

Liez un fichier dans votre dossier OneDrive pour synchroniser vos données sur tous vos appareils Windows.

🚀 Comment ça marche :
1. Cliquez "Lier un fichier OneDrive" ci-dessous
2. Naviguez vers votre dossier OneDrive et nommez le fichier
3. Sur vos autres appareils : cliquez ☁️ → "Charger depuis OneDrive"
