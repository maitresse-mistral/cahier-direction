Générateur de courriers
Sorties Scolaires

Mode d'emploi — École Cantarelle

Cet outil permet de créer en quelques clics des courriers pour les parents lors de sorties scolaires. Deux modes sont disponibles : sortie unique (une seule date) ou sortie récurrente (piscine, médiathèque…). Vous pouvez générer plusieurs courriers à la fois et imprimer le tout en orientation paysage, 2 sorties par page.

📋 Pour toutes les sorties

1

Choisissez le type de sortie

En haut de chaque fiche, sélectionnez :

📅 Sortie unique pour une sortie avec une seule date.

🔁 Sortie récurrente pour plusieurs séances (ex. piscine tous les jeudis).

2

Remplissez les informations

Pour chaque sortie : école, classe, destination, objet. Le champ Message complémentaire est facultatif.

3

Choisissez le signataire

Sélectionnez dans le menu : La directrice Le directeur L'enseignante L'enseignant

4

Activez les options du coupon

Accompagnateur cases à cocher pour les parents volontaires
Pique-nique ajoute une phrase d'info dans le courrier
Allergie ligne à remplir dans le coupon
Urgence numéro de téléphone à fournir le jour J

5

Ajoutez d'autres sorties si besoin

Cliquez sur ＋ Ajouter une sortie. Chaque fiche est rétractable (clic sur l'en-tête). Le bouton ✕ supprime une fiche.

6

Générez et imprimez

Cliquez sur 📄 Générer le document, puis 🖨️ Imprimer. L'impression se fait en paysage A4, 2 sorties côte à côte. Chaque sortie reste entière, sans coupure. Les parents rapportent la feuille entière signée.

🔁 Mode sortie récurrente

1

Ajoutez les dates des séances

Cliquez sur ＋ Ajouter une date pour chaque séance. Saisissez la date, l'heure de départ et l'heure de retour. Si toutes les séances ont les mêmes horaires, ils seront automatiquement mutualisés dans le courrier.

2

Le coupon s'adapte automatiquement

Si l'option Accompagnateur est activée, le coupon affiche une case à cocher par date — les parents cochent les séances où ils sont disponibles (plusieurs choix possibles).

💡 Bon à savoir

✏️ Vous pouvez modifier les informations et regénérer autant de fois que vous voulez avant d'imprimer.
🚌 Chaque sortie peut avoir des options différentes et un signataire différent.
📋 Le coupon contient toujours : nom, prénom, classe et nom du parent, plus les options activées.
Astuce : Si vous n'avez qu'une seule sortie, saisissez-la deux fois en cliquant sur ＋ Ajouter une sortie et en remplissant les mêmes informations. Les deux courriers s'impriment côte à côte sur une seule feuille A4 — il suffira de la couper en deux pour distribuer un exemplaire à chaque élève.
